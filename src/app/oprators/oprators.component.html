    Operators are functions. it has two types
  1. Pipe able Operator
  2. Creation Operator

Pipe able Operator is a function that takes an Observable as its inpu and returns another Observable. It is a pure operation: the previous Observable stays unmodified.

Creation Operators are the other kind of operator, which can be called as standalone functions to create a new Observable.

Common Operators:

  1. of, from => used to convert array into observable
  2. range => Creates an Observable that emits a sequence of numbers within a specified range.
  3. distinct => Returns an Observable that emits all items emitted by the source Observable that are distinct by comparison from previous items.
  4. first => Emits only the first value(or the first value that meets some condition) emitted by the source Observable.
  5. last => Returns an Observable that emits only the last item emitted by the source Observable.
  6. filter => Filter items emitted by the source Observable by only emitting those that satisfy a specified predicate.
  7. skip => Returns an Observable that skips the first count items emitted by the source Observable.
  8. take => Emits only the first count values emitted by the source Observable.
  9. toArray => Collects all source emissions and emits them as an array when the source completes.
  10. map => The function to apply to each value emitted by the source Observable.(just like for loop)
  11. tap => Used to perform side-effects for every emission on the source observable
  12. concat => concat joins multiple Observables together, by subscribing to them one at a time and merging their results into the output Observable.
  13. merge => Creates an output Observable which concurrently emits all values from every given input Observable.
  14. zip => It brings together two or more sequence of corresponding values as a tuple.
